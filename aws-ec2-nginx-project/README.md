# AWS EC2 Nginx Web Server Project

## Project Overview
Built and deployed a web server using AWS EC2 and Nginx.

## What I Did
- Launched an EC2 instance using AWS
- Configured Security Groups
- Opened Port 22 (SSH)
- Opened Port 80 (HTTP)
- Installed and configured Nginx on Ubuntu Linux
- Connected to EC2 using SSH

## Skills Used
- AWS EC2
- Ubuntu Linux
- Nginx
- Security Groups
- SSH
