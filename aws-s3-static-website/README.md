## Project Overview

This project demonstrates how to host a static website using Amazon S3.

The website was configured with static website hosting, bucket policies, and public access settings to make it accessible through the internet.

---

## AWS Services Used

| Service | Purpose |
|---------|---------|
| Amazon S3 | Host static website |
| Bucket Policy | Allow public access |
| Static Website Hosting | Serve HTML, CSS, and JavaScript files |

---

## Implementation Steps

1. Created an S3 bucket.
2. Uploaded website files.
3. Enabled Static Website Hosting.
4. Configured bucket policy.
5. Verified website accessibility.
